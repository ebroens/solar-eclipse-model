Solar Eclipse ‘light curve’ Models
==================================

This documentation provides a comprehensive guide to using the Solar Eclipse Model,
a tool for modeling solar eclipse light curves. The models are based on the
`BATMAN transit model`_ and include advanced features such as atmospheric extinction.
The source code for Solar Eclipse Models is openly available on `GitHub`_.
